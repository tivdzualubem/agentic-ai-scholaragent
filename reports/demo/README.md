# ScholarAgent Demo Screenshots

These screenshots document the Streamlit demonstration interface.

- `demo_input_page.png` shows the scholarship query and structured student profile.
- `demo_verified_result.png` shows a completed verified-fallback result with ranked evidence and citations.
- `demo_abstention.png` shows safe abstention when no sufficiently verified scholarship evidence is retrieved.

The screenshots use the fast verified demonstration mode. This mode performs
deterministic citation verification and fallback without external LLM calls.
